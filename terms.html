Privacy and Data Policy

Copyright 2023-2025 MAR2XGAMES – PRIVACY & DATA PROTECTION POLICY
Marmargee vs. Monsters is the abbreviated game title of the game Marmargee Fighter Girl vs. Zombies & Monsters!
Marmargee Fighter Girl and Matmateo Fighter Boy are the main characters in this game.

Privacy policy and data protection compliancy:
we do not collect or store any private data of our users other than what is necessary and possible via the publishing platforms to inform the users of the game in case of important changes, new releases or to able to improve the game experience. It is possible that the platforms on which this game is published do collect more private data (for example for advertisement purposes) than is listed here below. Any misuse of this information is unknown to MAR2XGAMES and beyond the control of MAR2XGAMES.

Specifically this version of the game uses cloud services from Azure PlayFab* in which the log in data of the user is stored: these are (but in the future may not necessarily be limited to) the email address (if provided the option) and/or Google Play id ** and/or Steam id *** of the user (this service does not store any passwords in a way that is readable to MAR2XGAMES).
Additionally the following data is stored/updated each time the user plays the game:

  • The date and time of first registration and last login date and time
  • In a generic way the country and city of login
  • The save game data (game progress) of the user
    This data can be used to draw generic statistics like the number of logins or play duration.

Login data or save game data may be shown in the cloud functionalities of PlayFab like for example leaderboards.
The game may contain features like taking in-game screenshots: these are stored only on the local machine of the player. It is responsibility of the player to share them correctly on social media. Any negative side effects of such sharing is beyond the control of MAR2XGAMES.

The data will not be stored longer than is required to ensure correct game functionality. The main goal of this storage is allow users to log in seamlessly across devices over an extended period of time while keeping their saved game progress data even when the game is removed from a device. Each platform used is to be considered a separate game with a separate save file (the game played on MS Store is not the same as the game on Steam and is not the same as the one on Google Play or any other future platform on which the game may be published).

Removing the game in itself from a device will not remove player save game data in itself.
In the policy of MAR2XGAMES a 5 year period is given to users that do not log in to play the game after which all data stored in the cloud may be removed (without guarantee of notification).

REMOVAL PROCEDURE of your data:
The player can erase player name and all other saved game progress data within the game (used when making screenshots or sharing on social media) via the setup menu and choosing to reset the game. This does not erase the account data in the cloud. This function does not erase screenshots stored on the devices and can not erase shared screenshots on social media: it is the responsibility of the player to erase those from the devices or from the social media platforms directly.

Any player has the right to have his/her account related data removed entirely from the cloud by sending a request to the following email address specifying clearly which userid or email address should be removed (note that this will remove all related save game data): mar2xgames@gmail.com
It is possible the player may have to answer questions before this removal takes place to avoid abuse of this procedure.

** A Google Play account is a product of Google
*** Steam is a product of the Valve corporation
